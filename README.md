# node-random-image

一个超级简单 Node 服务（31 行代码），随机返回一张图片

## Api

刷新页面你将看到不一样的图片 or 复制链接在浏览器中打开

## Usage

```sh
https://wcao.cc/r/a/{type}

# or

https://wcao.cc/image-space/api/{type}
```

## Install

```bash
git clone https://github.com/meetqy/image-space
yarn
yarn start
```
